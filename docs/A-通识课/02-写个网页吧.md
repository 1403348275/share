### 网页的皮肉

> 来写样式表吧

```css
body {
  /* 内边距 */
  padding: 10vh 10vw;
}

h1 {
  /* 文字大小 */
  font-size    : 3em;
  margin-bottom: .8em;
}

h2 {
  font-size    : 2em;
  margin-bottom: 1.4em;
}

p {
  font-size  : 1.4em;
  /* 字体粗细 */
  font-weight: 300;
  /* 行高 */
  line-height: 1.6em;
  /* 最大宽度 */
  max-width  : 600px;
}
```

### 动态的网页

> 来写js吧

```js
$.getJSON('info.json', function (json) {

  $('#name').text(json.name)
  $("#intro").text(json.intro)
  $("#about").text(json.about)

  // 0为女生，1为男生
  if (json.sex) {
    $("#sex").text('👦')
  } else {
    $("#sex").text('👧')
  }
  
});
```
